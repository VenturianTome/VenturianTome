# VenturianTome

A fanmade library of lore and trivia for the characters of the YouTube channel VenturianTale  

This uses [mdbook](https://rust-lang.github.io/mdBook/) for organization & compsition

To build:
```
cd VenturianTome
mdbook build -o
```

To view:
```
cd VenturianTome
mdbook serve
```
and open http://localhost:3000/
