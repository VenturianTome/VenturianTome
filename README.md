# VenturianTome

A fanmade library of lore and trivia for the characters of the YouTube channel VenturianTale  

This uses mdbook for organization & compsition

To build:
```
cd VenturianTome
mdbook build
```

To view:
```
cd VenturianTome
# optional
mdbook watch
python3 -m http.server
```
and open http://localhost:8000/
